g2's PEAR Channel Server

Registering the channel: 
pear channel-discover gomboc.github.com/pear
Listing available packages: 
pear remote-list -c g2
Installing a package: 
pear install g2/package_name
Installing a specific version/stability: 
pear install g2/package_name-1.0.0
pear install g2/package_name-beta
Receiving updates via a feed: 
http://gomboc.github.com/pear/feed.xml
